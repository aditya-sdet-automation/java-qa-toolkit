# Java QA Toolkit
A collection of Java based utility toolkit and automation scripts built for software Quality Assurance (QA) and testing.

## Current Tools
* **Bug Validator:** A dynamic command-line tool that captures bug titles, verifies reproducable steps, calculates defect severity, and generates formatted reports. Built with robust defensive programming to handle hostile or blank user inputs without crashing.

* ## Technical Skills Demonstrated
* * Defensive Programming & Input Validation
  * Modular Code Architecture (Manager/Worker method isolation)
  * Java 'Scanner' buffer management & logic flow control
